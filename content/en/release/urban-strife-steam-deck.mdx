<Section icon={Swords} title="Steam Input Configurations for Tactical Controls">

The default user interface of Urban Strife relies heavily on mouse hover states and keyboard hotkeys. This design makes controller navigation difficult without custom Steam Input mapping. By utilizing the Steam Deck trackpads and back buttons, you can create a layout that mimics a native controller setup.

Mapping these controls allows you to quickly select units, manage inventory, and target specific body parts during combat. Follow the steps below to configure your custom controller layout.

<Steps>
  <Step num="1" title="Access Controller Settings">
    Press the Steam button while the game is running, navigate to the right, and select Controller Settings.
  </Step>
  <Step num="2" title="Map Right Trackpad to Mouse">
    Set the Right Trackpad behavior to As Mouse. Adjust the sensitivity to medium to allow for precise targeting of zombie limbs.
  </Step>
  <Step num="3" title="Create Left Trackpad Radial Menus">
    Configure the Left Trackpad as a Radial Menu. Map the buttons to keyboard shortcuts like I for Inventory, M for Map, and C for Character Sheet.
  </Step>
  <Step num="4" title="Assign Back Paddles">
    Map L4 and R4 to Quick Save (F5) and Quick Load (F9). Map L5 and R5 to cycle through your squad members (Tab and Shift+Tab).
  </Step>
</Steps>

<Callout type="info" title="Control Scheme Recommendation">

Managing hunger, wounds, and ammo is core to the gameplay. When large hordes move simultaneously, the Steam Deck CPU faces heavy calculations, causing temporary framerate drops. Optimizing the game's background processes can help mitigate these performance dips.
